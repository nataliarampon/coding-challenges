# coding-challenges
Coding exercises using different data structures and programming paradigms to practice programming for fun.

## Pre-requisites
- node.js `v20.9.0^`
- npm `v10.1.0^`
- python`v3.9.6^`


## Running Python Tests
```bash
python3 -m unittest discover -s **/py 
``` 

## Running Typescript Tests
```bash
npm run test
```
